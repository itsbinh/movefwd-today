## App Name

| Field | Value |
|-------|-------|
| **Name** | Movefwd (move forward) |
| **Domain** | movefwd.today |
| **Tagline** | "Your next step, simplified" |
| **Alt Domains** | movefwd.life, movefwd.app |
| **Vibe** | Empowering, forward-looking, universal |

### Name Rationale
- **Move forward** — Progress, hope, dignity. Not "needy person gets help" — "I'm getting somewhere."
- **.today** — Urgent, immediate action. Not "someday" — today.
- Universal: works for college students, families, people in crisis, anyone wanting to progress
- Destigmatized: doesn't sound like "charity app" — sounds like a tech product

## Logo

### Concept: Magnifying Glass + Heart

**Design:**
- Simple magnifying glass shape
- Inside the glass: a small heart or checkmark
- Color: Primary coral gradient
- Clean, scalable — works at 32x32px

**Alternative: Location Pin + Check**

- Map pin shape
- Inside: checkmark (found it!)
- Color: Primary coral

### Logo Usage

| Size | Clear Space |
|------|-------------|
| Full logo | 24px minimum |
| Icon only | 16px minimum |

### Don't Do
- Don't stretch or distort
- Don't use on busy backgrounds
- Don't add effects (shadows, gradients unless brand gradient)

## Iconography

### Style
- Filled icons (not outline)
- Rounded corners
- 24px standard size
- Match category colors above

- **Phosphor Icons** (phosphoricons.com) — Clean, modern, free

## Motion & Animation

### Principles
- Subtle, not flashy
- 200-300ms duration
- Ease-out timing

### Examples
- Page transitions: Fade + slight slide up (200ms)
- Button hover: Scale 1.02
- Cards: Fade in on scroll
- Map markers: Gentle bounce on appear

## Voice & Tone

### In Copy

| Instead of... | Use... |
|---------------|--------|
| "Homeless shelter" | "Emergency housing" |
| "Assistance" | "Resources" |
| "Apply" | "Get started" |
| "Low income" | "Anyone can use" |
| "Charity" | "Support" |
| "We help you" | "You deserve support" |
| "Unfortunately" | — (avoid negative language) |

### Tone
- **Empowering**: Focus on what user CAN do
- **Warm**: Friendly, not corporate
- **Clear**: Simple words, short sentences
- **Inclusive**: Works for everyone
